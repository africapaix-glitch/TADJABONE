# Code of Conduct

Tous les contributeurs de TADJABONE sont tenus de respecter le présent Code of Conduct. Le but est d'assurer un environnement accueillant, respectueux et collaboratif.

Exemples de comportements acceptables :
- Être respectueux et poli dans les discussions.
- Accepter des retours constructifs et fournir des critiques bienveillantes.
- Aider les nouveaux contributeurs à s'orienter.

Comportements inacceptables :
- Harcèlement, discours haineux, propos discriminatoires.
- Attaques personnelles, insultes ou menaces.
- Partage de contenu illégal ou sexuellement explicite.

Procédure de signalement :
- Si vous êtes témoin ou victime d'un comportement inapproprié, envoyez un message privé à : africapaix-glitch
- Fournissez des détails (liens, captures d'écran, descriptions) pour permettre une enquête.

Conséquences :
- Les mainteneurs se réservent le droit de retirer l'accès, supprimer des commentaires, ou exclure des contributeurs en cas de non-respect répété.

Remarque : Ce document s'inspire des bonnes pratiques communautaires et peut être mis à jour.
